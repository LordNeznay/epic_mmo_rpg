2D-игра жанра MMORPG.
=====================

Выполняется в рамках проекта "Технопарк" по предметам "Фронтенд" и "Углубленное программирование на Java" студентами Курулёвым Андреем и Каменским Валентином.

*Для запуска нужно:*

* В корневой папке выполнить команду `grunt`
* Если `grunt` ругается на отсутствие пакетов, нужно выполнить команду `npm install`
* Открыть в браузере (желательно в Chrome) адрес `http://localhost:8080/`


*Управление*

* WASD - передвижение по карте
* Z - захватить флаг (при условии, что отображается возможность нажатия)
* ПКМ - выбрать цель
* 1 - Ударить цель
