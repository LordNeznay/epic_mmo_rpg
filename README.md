2D-игра жанра MMORPG.
=====================

Выполняется в рамках проекта "Технопарк" по предметам "Фронтенд" и "Углубленное программирование на Java" студентами Курулёвым Андреем и Каменским Валентином.

*Для запуска нужно:*

* В корневой папке выполнить команду `grunt`
* Если `grunt` ругается на отсутствие пакетов, нужно выполнить команду `npm install`
* Открыть в браузере (желательно в Chrome) адрес `http://localhost:8080/`

*Краткое описание игры*

Для начала игры необходимо 2 игрока. Сражение между двумя командами за флаг. Игроки имеют возможность атаковать врагов и лечить союзников. Игра заканчивается, когда какая-то из команд наберет N очков (по 1 за секунду удержания флага).

*Управление*

* WASD - передвижение по карте
* Z - захватить флаг, при условии, что отображается возможность нажатия (находится в радиусе одной клетки от флага)
* ПКМ - выбрать цель
* [0-9] - соответствующая способность (см. панель)
